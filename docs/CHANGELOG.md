### v0.2.0 - 用户管理功能（卡片015）
**日期**：2026-03-25
**状态**：已完成

#### 新增
- 实现用户与权限管理功能（卡片015）
  - 后端：用户CRUD接口、密码加密（bcrypt）、登录接口
  - 前端：用户列表页、搜索筛选分页、新增/编辑弹窗
  - 数据库：users表（符合DB-STANDARD规范）
- 创建11个测试账号（涵盖7个角色）
- 创建默认管理员账号 admin/admin123
- 新增一键启动脚本 `./start`
  - 支持 `./start` 启动前端
  - 支持 `./start backend` 启动后端
  - 支持 `./start all` 同时启动前后端
  - 支持 `./start stop` 停止所有服务

#### 技术细节
- 后端：FastAPI + SQLAlchemy + SQLite
- 前端：React 19 + Ant Design 5.x
- 密码加密：bcrypt
- 后端API：
  - POST /api/users - 创建用户
  - GET /api/users - 用户列表（分页、搜索、筛选）
  - PUT /api/users/{id} - 更新用户
  - DELETE /api/users/{id} - 删除用户（软删除）
  - POST /api/auth/login - 用户登录
