## 功能：登出功能

### 1. 这个功能是干嘛的？
让用户安全退出系统，清除本地登录状态（token和用户信息），并返回登录页面。

### 2. 页面上有什么？

#### 登出入口位置
在**主布局的顶部栏右侧**，用户头像/名称旁边：
- 用户头像（圆形）
- 用户真实姓名
- 下拉菜单箭头
- 点击后展开下拉菜单

#### 下拉菜单内容
- 用户信息展示（姓名 + 角色）
- 分隔线
- "退出登录"选项（带退出图标）

### 3. 用户怎么操作？

#### 登出流程
1. 用户点击顶部栏右侧的用户头像/名称
2. 展开下拉菜单
3. 点击"退出登录"选项
4. 弹出确认对话框："确定要退出登录吗？"
5. 点击"确定"：
   - 清除localStorage中的token
   - 清除localStorage中的userInfo
   - 可选：调用后端登出接口（记录审计日志）
   - 提示"已退出登录"
   - 跳转到登录页面
6. 点击"取消"：关闭对话框，保持在当前页面

### 4. 点完按钮发生什么？

#### 登出成功场景
- 清除localStorage：token和userInfo被删除
- 显示提示："已退出登录"
- 页面跳转：自动跳转到 /login
- 无法回退：后退按钮不会回到已登录状态（因为token已清除）

#### 取消登出场景
- 关闭确认对话框
- 页面无变化
- 用户保持登录状态

### 5. 什么样算做好了？（验收标准）

#### 页面展示
- [ ] 顶部栏显示当前登录用户头像和姓名
- [ ] 点击用户区域展开下拉菜单
- [ ] 下拉菜单中有"退出登录"选项

#### 登出功能
- [ ] 点击"退出登录"弹出确认对话框
- [ ] 确认后清除localStorage中的token
- [ ] 确认后清除localStorage中的userInfo
- [ ] 确认后提示"已退出登录"
- [ ] 确认后自动跳转到登录页面
- [ ] 取消后保持在当前页面

#### 安全验证
- [ ] 登出后点击浏览器后退，不会显示登录后的页面
- [ ] 登出后访问需要登录的页面，自动跳转到登录页
- [ ] 登出后无法操作任何需要权限的功能
